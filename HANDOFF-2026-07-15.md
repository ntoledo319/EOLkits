## 8. CRITICAL GOTCHAS & LESSONS (the landmines that cost time this session)
1. **AWS DATES — THE #1 TRUTH LANDMINE.** Blogs, the workflow synthesis, and even a prior AWS re:Post article cite
   **superseded** dates (e.g. nodejs20 "Aug 31 / Sep 30 2026"; python3.9 "Jan 15 / Feb 15 2026"). **AWS DELAYED the
   blocks** into a synchronized **Q1-2027 cluster.** The authoritative source is the AWS Lambda runtimes table
   (docs.aws.amazon.com/lambda/latest/dg/lambda-runtimes.html): nodejs16/18/20 + python3.8/3.9/3.10 → **block-create
   2027-02-01, block-update 2027-03-03**; python3.11 → 2027-07-31 / 2027-08-31; deprecation (patches-stop) dates
   differ per runtime (nodejs20 = 2026-04-30, python3.9 = 2025-12-15). AL2 OS EOL = **2026-06-30 (PAST, now unpatched)**.
   `rules/public/deprecations.yml` is CORRECT and drives the site — verify against it + the AWS table before ANY date
   claim. When writing public answers, **hedge** ("per the AWS table … AWS has revised these before") and stress "it's
   already unpatched, migrate now regardless."
2. **GIT: the routine pushes concurrently.** Always `git fetch && git pull --rebase origin marketing-machine-v2`
   before pushing. If a rebase conflicts (usually both sides appended to `DECISIONS.md`/`METRICS.md`), **keep BOTH**
   sections (renumber your decision) — never drop the routine's work. This happened this session; resolved by keeping
   both and renumbering D11/D12.
3. **`docs/` is a STALE build artifact.** It's tracked but the box **rebuilds it from source** on deploy. **Commit
   SOURCE only** (`apps/web/build.py`, `apps/web/content/*.yml`, `apps/web/templates/*`, `rules/public/deprecations.yml`);
   after a local build, `git checkout -- docs/` and `rm -rf` any new `docs/` artifacts before committing. Adding a page
   = new `build_*` fn + one line in `main()`'s pages dict + one line in `sitemap.xml.j2`. Keep it byte-deterministic
   (no `datetime.now()` in output — compute countdowns client-side); the determinism test enforces this.
4. **The $1,499 Pack money-path was reviewed (DECISIONS D9).** Fixed a critical bug (a Pack charged with a blank
   `installation_id` delivered nothing AND skipped the auto-refund) + added a test. **Still-open PRE-SALE GATES (do
