# EOLkits - AWS Deprecation Scanner

EOLkits scans CloudFormation, SAM, Terraform, JavaScript, TypeScript, Python, and JSON files for AWS runtime and platform deprecation risk.

## Commands

- `EOLkits: Scan Workspace for Deprecations`
- `EOLkits: Show Deprecation Report`
- `EOLkits: Get Full Audit Report`

## Settings

- `eolkits.enabledKits`
- `eolkits.severityThreshold`
- `eolkits.autoScan`

The extension runs locally in VS Code and links to the hosted audit flow for deeper reporting.
